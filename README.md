
# 🛒 Sistema de Controle de Vendas — Java

Projeto desenvolvido em Java para praticar os fundamentos de **Programação Orientada a Objetos (POO)** utilizando apenas recursos básicos da linguagem.

## 🚀 Funcionalidades

- Cadastro de produtos
- Listagem de produtos cadastrados
- Venda de produtos com validação de estoque
- Cálculo do total vendido
- Menu interativo no terminal

## 🧠 Conceitos aplicados

- Encapsulamento (`private`, getters e setters)
- Construtor
- Métodos com regra de negócio
- Vetor de objetos
- Estruturas de repetição (`while`)
- Estrutura de decisão (`switch`)
- Uso do `Scanner`
- Separação de responsabilidades entre classes

## 🏗️ Estrutura do Projeto

- `Produto` → Representa o produto e gerencia o estoque
- `Loja` → Controla o sistema e o fluxo do menu

## ▶️ Como executar

1. Compile as classes do projeto
2. Execute a classe `Loja`
3. Utilize o menu pelo terminal

---

Projeto focado em fortalecer a base de orientação a objetos antes de avançar para conceitos como herança e coleções.
